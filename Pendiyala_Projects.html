Scratch Project : Detective Game

Date

October 30, 2018 - September 8, 2018 (9 days)

Class

Computer Science Principles (CSP)

Members

Kaushik Pendiyala and Siri Phaneendra

Skills Gained from Project

Teamwork and Resiliance

Synopsis of Project

This was a project in which my partner, Siri Phaneendra, and I coded an interactive, thrilling murder mystery. As a well known detective, the users job is to figure out who robbed the bank the night before. The user is left with a set of clues, each leading them down a different path, eventually determining if they will catch the criminal or not. My role in this project was mainly to focus on the coding, as well as contributing ideas for our game. My partner, Siri, focused on the documentation of our project, which explained every decision we made throughout the development of our game. Although there were many problems that we encountered throughout our project, Siri and I used teamwork to solve them. Siri and I worked very well as a team, as we both implemented new ideas and were open to suggestions. One significant obstacle that we encountered was when our Scratch game code broke just as we were nearing the deadline. Siri and I used teamwork to solve this problem, as she handed out suggestions for what might be wrong, while I fixed the code to test for different problems. We soon found the problem and fixed it effortlessly because of our teamwork and resilience.

Python Project : Image Manipulation

Date

February 26, 2019 - March 28, 2019 (31 days)

Class

Computer Science Principles (CSP)

Members

Kaushik Pendiyala and Jericho Lim

Skills Gained from Project

Teamwork and Self-Learning

Synopsis of Project

In this project, my partner and I were prompted to create a portrait for a family picture that included various customizable parameters. We decided to include rounded edges, a border, greyscale, a logo, and text in our customizable family portrait. Throughout the creation of our portrait, Jericho and I communicated with each other effectively in order to make sure that both of us understood all parts of our code. Whenever I had a question, Jericho would explain it to me in detail, making sure I fully comprehended the code before moving on. This way, both of us understood what each line of code does in our program, consequently deepening our knowledge on image manipulation. We also worked on the documentation of our project, which explained our code and displayed all our images in a concise, effortless way. Even with the communication between Jericho and I, we still encountered some problems that we were not able to solve ourselves. In order to solve these problems, we had to turn to other reference documents on the internet and learn from these sources. By combining our prior knowledge on PIL with our teamwork, we were able to find solutions to most of our problems regarding the code quickly and effortlessly. Overall, this was a very fun and fascinating project in which both Jericho and I used our teamwork and self-learning skills to create a customizable family portrait that the user would love.

Members

Synopsis of Project

In this project, my partners and I created an educational game that has multiple levels represented by an era in human civilization. For example, the first level would be the bronze age, the second level would be the stone age, the third level would be the iron age, and so on and so forth. To progress to the next level, the user would have to complete a set of history themed trivia minigames. In these minigames, the user would have to answer a set of questions before bears attack your villiage and deplete your health bar. If you sucessfully complete the mini-game, you would move on to the next level but if you didnt, you would have to restart the game. My specific role in this project was to asssist and delegate tasks to everyone on their parts while doing my own. I helped mainly with the implementatioon of the questions into the minigame and helped my parters with their work too. I took the initiative with the coding and developed a lot of leadership skills throughout this project.
